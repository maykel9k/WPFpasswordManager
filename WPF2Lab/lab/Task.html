Password Manager

Your task is to create a base for simple password manager application. It will allow user to store images and account information for various websites. User data can be organized into folders. This is a first part of WPF Task for 12 points. In case of any doubts concerning the task, please inspect the sample application or consult your teacher.

Laboratory Part

  • Sample Application

  • Main window:
    • Title: "Password Manager"
    • Initial size: 1200x600
    • Minimum size: 600x300
    • Opens in the center of the screen

  • Login screen:
    • Content in the center of the screen
    • "Password" text above Password input box and "Unlock" Button
    • Pasword input masks password
    • Unlock button switches application to the main screen

  • Main screen:
    • Menu with items aligned to the right with font size 14
    • "Save" "Logout" and "Password Generator" menu options
    • Logout switches application to the Login screen
    • Screen divided into left and right parts, right is always 4 times larger than the left

  • Left part:
    • Tree of folders and files
    • Right click opens context menu. Options to add directories, password files and image files with unique names
    • Right click on directories opens context menu. It displays directory name in the top. Option to add chilld directories to directories
    • Directories are displayed with bold font and files with italic

  • Right part:
    • Changes depending on the element selected in the tree
    • For selected directories it displays directory name and number of elements in this directory
    • For selected images it displays TopSecret.png image
    • For selected passwords it displays password editor

  • Password editor:
    • Editor diided into left and right parts
    • Parts can be resized. Minimum widths of the left and right parts are 150 and 300
    • List of passwords on the left. List has no horizontal scroll bar and vertical scroll bar displays only if needed
    • List items show password image, name, login and colored strength of the password. Each strength level has unique color. Password strength should display to the right of an item. Use PasswordStrengthUtils.CalculatePasswordStrength from PasswordStrength.cs
    • Button to add new passwords

    • Password form to the right. Setting password icon, name, email, login, password, website and notes
    • Email, login and passwords have "Consolas" font
    • Notes input has multiple lines of text with minimum of 5 and maximum of 10 lines
    • Under the password there is a progress bar that shows strength of the password and is colored with the same color as password strength in the list. It updates during editing

    • Use converters for password strength text, colors and progress bar
  • Tips:
    • Frame, Page, TreeView, ContentControl, DataTemplate, GridSeparator, INotifyPropertyChanged, IValueConverter, ICommand
  • Points:
    • Main window and login screen 1p
    • Main screen menu and changing between screens 1p
    • Tree of folders and files 2p
    • Adding directories to directories 1p
    • Changing right part depending on selected item 2p
    • List of passwords 2p
    • Password form and password editor resizing 3p
